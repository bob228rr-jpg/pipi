# $PIPI · Pink Pepe's Girlfriend

Pink Pepe's Girlfriend: a cute, pink, feminine Pepe-coded meme coin landing page.

## About

- **Name:** Pink Pepe's Girlfriend
- **Ticker:** $PIPI
- **Style:** pink meme cartoon, bold HORPE-inspired layout, generated website assets

> Bows, blush, bags, and pink candles.

## Stack

Single-file static site. `index.html` with inline CSS and JS, assets in `./assets/`. No build step, no dependencies. Works on GitHub Pages out of the box.

## Local

Open `index.html` directly in a browser, or serve locally:

```bash
python3 -m http.server 8000
```
